PicGo+GitHub搭建图床

2023-05-25
学习
字数统计: 445   |   阅读时长≈ 1 分钟

前言

在写文章时经常会需要插入一些图片，本地图片链接又略显不方便，这时就需要图床了。图床是互联网中存储图片的空间，是将图片上传到一个服务器，然后生成一个访问链接，链接地址指向图床。
现将个人搭建图床的过程进行分享！

安装

下载PicGo

PicGo：一个用于快速上传图片并获取图片 URL 链接的工具。
目前支持七牛图床、腾讯云COS、有拍云、GitHub、SM.MS和阿里云OSS。
下载地址：

下载源 地址/安装方式 支持平台
GitHub https://github.com/Molunerfinn/PicGo/releases All
腾讯云COS https://github.com/Molunerfinn/PicGo/releases All
山东大学镜像站 https://mirrors.sdu.edu.cn/github-release/Molunerfinn_PicGo All
Scoop scoop bucket add helbing https://github.com/helbing/scoop-bucket & scoop install picgo Windows
Chocolatey choco install picgo Windows
Homebrew brew install picgo --cask macOS
AUR yay -S picgo-appimage Arch-Linux

下载完成后安装即可。

GitHub设置

  1. 登陆 GitHub
  2. 新建仓库

  3. 生成token
    点击GitHub个人头像, 点击Settings，再点击Developer settings，选择Tokens(classic)


    点击Generate new token

    把repo的勾打上, 然后到最底部, 点击Generate token生成token

    复制token存到其他地方以备以后要用。

配置PicGo

按照下图配置PicGo
GitHub仓库名的格式是用户名/仓库，一般我们选择main分支，点击确定以生效，可以点击设为默认图床来确保上传的图床是GitHub。

结语

到此为止，图床配置完毕，上传图片，就会你的仓库里也会增加新的图片，其他配置详见PicGo官方。
